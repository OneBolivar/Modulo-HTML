**CARACTERISTICAS**
- Este sitio web esta hecho en la estructura HTML, apoyandome con el lenguaje CSS, integrando el SASS para optimizar el flujo de estilos y JS para las animaciones.
- Tiene un diseño responsive.
-  
**TECNOLOGIAS USADAS**
- HTML5: Estructura semántica del contenido.
- CSS3: Estilos avanzados y layouts.
- Sass: Preprocesador para una arquitectura de estilos modular y mantenible.
- JavaScript : Lógica del lado del cliente e interactividad.
